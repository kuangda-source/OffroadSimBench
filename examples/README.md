# Examples

Runnable smoke tests and demos:

- `check_backends.py`: print registry status for all backends.
- `run_gym_demo.py`: run a local GymHeightmapBackend episode.
- `run_world_model_agent.py`: run the first world-model agent baseline.
- `run_gymnasium_env.py`: step the Gymnasium wrapper with random actions.
- `run_dataset_replay.py`: replay a dataset through the adapter registry.
- `check_beamng_runtime.py`: inspect BeamNG auto-detection and optionally run a real connection smoke test.
- `run_mock_ue5_backend.py`: exercise the UE5 TCP JSON bridge with a local mock server.
- `replay_episode.py`: inspect a saved episode directory.
